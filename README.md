# مصروفاتي

تطبيق ميزانية شخصية مجاني بالكامل، مستقل عن مشاريع العمل، ويعمل محليًا على الجهاز.

## التشغيل السريع على ويندوز
1. فك الضغط.
2. افتح المجلد في Terminal.
3. شغّل:
   python -m http.server 8080
4. افتح المتصفح على:
   http://localhost:8080

> فتح index.html مباشرة يسمح باستخدام أغلب الوظائف، لكن التثبيت والعمل دون اتصال يحتاجان تشغيله عبر خادم محلي أو استضافته على GitHub Pages/Netlify.

## المزايا
- تسجيل الدخل والمصروفات.
- الالتزامات الشهرية وحالة السداد.
- أهداف الادخار.
- تقارير ونسب شهرية.
- تصدير CSV.
- نسخة احتياطية واستعادة JSON.
- حفظ محلي دون تسجيل دخول.
- مظهر فاتح وداكن.
- قابل للتثبيت كتطبيق PWA.

## الخصوصية
كل البيانات محفوظة داخل متصفح الجهاز باستخدام LocalStorage ولا تُرسل لأي جهة.
